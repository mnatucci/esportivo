# meta44 - Landing Page TikTok Shop

Landing page moderna e interativa para venda de seguidores brasileiros para TikTok Shop.

## 📋 Sobre

meta44 é uma plataforma que ajuda criadores de conteúdo a ativar seu TikTok Shop com seguidores brasileiros, permitindo que comecem a ganhar dinheiro como criadores e afiliados.

## 🚀 Como usar

### Opção 1: Abrir localmente
1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/meta44.git
cd meta44
```

2. Abra o arquivo `index.html` no seu navegador:
```bash
open index.html
# ou
start index.html
# ou
xdg-open index.html
```

### Opção 2: Hospedar no GitHub Pages
1. Vá para as configurações do repositório
2. Em "Pages", selecione a branch `main` como fonte
3. A página será publicada em: `https://seu-usuario.github.io/meta44`

### Opção 3: Usar um servidor local (Python)
```bash
python -m http.server 8000
# Acesse http://localhost:8000
```
